# Git/GitHub Guide

## Comandos básicos:

``` bash
git add file.txt # adiciona aquivo em staging
git commit -m "add file" # commita mudanças
git push origin master # envia para Git remoto
git pull origin master # puxa do Git remoto
git checkout <branch> # muda para contexto da branch <branch>
git checkout -b <branch> # cria branch <branch> e muda para o contexto dela
```
